# SQL Reference Object

A structured and easy‑to‑navigate SQL reference designed to help beginners and intermediate learners quickly look up essential SQL commands.  
This project mirrors the workflow from the FreeCodeCamp Git tutorial, but rebuilt and expanded in my own GitHub Codespace environment.

---

## 📘 Project Overview

This repository contains a JSON‑based SQL reference object that organizes common SQL operations into clear categories:

- **Database commands**  
- **Table commands**  
- **Row commands**  
- **Column commands**  
- **Constraints & references**

The goal is to provide a simple, readable, and extensible reference that can be used for learning, quick lookup, or integration into other projects.

---

## 🧩 Features

- Structured JSON reference for SQL commands  
- Includes `CREATE`, `DROP`, `INSERT`, `UPDATE`, and more  
- Clean organization by SQL operation type  
- Easy to extend with additional SQL syntax
